# Age-of-Empires-Crusher
an Age of Empires II AI

## Install
1. Install Age of Empires II
2. Install UserPatch v1.4
	http://userpatch.aiscripters.net/
3. set correct Age of Empires path in deploy.py
4. run deploy.py
